▸ Four Pillars

TWO-PHASE WARRANTS

Authorization before action. Attestation after. Audit trails begin before execution, not after. State machine: PENDING → AUTHORIZED → ATTESTED

PHYSICS ENGINE

Validate claims against hardware reality. 10GB in 1ms on 3.35TB/s bandwidth? Impossible. The agent is lying. Physics doesn't negotiate.

CHECKPOINT SURETY

Hash-chained records at every tool invocation boundary. Token attribution distinguishes agent decisions from environment responses. Tamper-evident by construction.

SEALED CONTEXT

Agents optimize. Safety is inefficient. They will learn to skip it. Procedural invariants enforced at governance layer, not agent layer. Cannot be optimized away.
